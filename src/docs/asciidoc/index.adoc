== 상태 코드 (Http Status)

[cols="3,3"]
|====
|Http Status | 설명

|`OK(200)` |요청이 성공적으로 완료 됨
|`BAD_REQUEST(400)`|서버가 요청의 구문을 이해할 수 없음
|`UNAUTHORIZED(401)`|요청이 사용자 인증을 필요로 함
|`FORBIDDEN(403)`|사용자 권한이 부족함
|`NOT_FOUND(404)`|요청한 리소스를 찾을 수 없음

|====
